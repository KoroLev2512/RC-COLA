# RC COLA Project

Landing page project for RC COLA using modern web technologies.

## Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling with Flexbox and Grid.
- **JavaScript (ES6+)**: Interactive elements and animations.

## Assets

All visual assets (logos, product images, backgrounds) are located in the `assets/` directory.

## How to Run Locally

You can run this project locally using several methods:

### Method 1: Open in Browser
Simply double-click `index.html` or drag it into your web browser.

### Method 2: Python HTTP Server (Recommended)
Run the following command in your terminal within the project directory:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.

### Method 3: WebStorm / IDE
If you are using an IDE like WebStorm, use the built-in "Open in Browser" feature to launch a local development server.
